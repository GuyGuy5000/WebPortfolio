Welland Minor Baseball League

WMBA was a collaborative project commisioned by the Welland Minor Baseball League to manage and track baseball teams and games. Coded as a .NET Web App, this project utilized agile methodology and was separated into three sprints. It uses an SQLite database with MVC design and ASP.NET Identity to authenticate user roles.

My role was communications lead and lead developer. This included responsibilities such as managing incoming code using GitHub and documenting all aspects of the projects. Along with managing the project, I created data models and programed entire controllers and their corresponding views.

Calendarium

Calendarium is a cross-platform calendar tracking application designed for organizing and scheduling different facets of life. This app was created in .NET MAUI and uses XAML for UI design. The backend integrates a custom built ASP.NET Core Web API using MVVM (Model-View-ViewModel) design patterns to ensure separation of logic and UI.

Arkanoid in UWP

The goal of this school project was to recreate Arkanoid to gain a better understanding of software design process such as documentation, class diagrams, and refactoring code. Additionally, the .NET FileIO class was used to read and write a user's highscores to a csv file
