Calendarium

Discover Lincoln is a website designed to attract tourism to the town of Lincoln. This site was built using HTML, CSS, NodeJS, and ReactJS. Designed to be quickly and efficiently maintained using a Content Management System, the site uses an API to load new and updated content.

With a responsive and intuitive design, the site places emphasis on being user friendly and accessible by following W3C accessibility guidelines. This site includes an interactive map page built using Mapbox. Providing the user with a search and filter function to create an effortless user experience.

Catering Management Solution

Emma's small engines was a collaborative school project build to be employee facing and used to streamline customer orders and repairs. Created with HTML, CSS, and JavaScript. Made using agile methodology, this project was completed in three sprints. Throughout these sprints, my role was to lead communications and coordinated team members. Additionally I coded whole pages, custom dropdowns, and context help using the mentioned programming languages to ensure W3C guidelines were followed.

The site contains pages for maintaining a database and generating reports. There is also a help feature to enhance user experience and simplfy onboarding for new employees. The site also implements user roles by restricting access to non-admin users, and allowing full access to users with authorization.
