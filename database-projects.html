Contoso Reporting Dashboard

This Contoso dynamic reporting dashboard was build using Power BI. It uses data housed in an SQL Server Data Warehouse retrieved from a demo dataset (created by Microsoft). The data model was trimmed down to only the relevant tables and columns to efficiently read data and simplify aggregations/calculations. Emphasis is placed on size and positioning of data as well as using effective visualizations to create an informative picture.

Catering Management Solutions

Catering Management Solutions is a school project designed to organize a hotel management system for events and functions. This project was build as an ASP.NET Core Web App and utilizes MVC design patterns to separate logic from UI. It features a complex relational database built with a code first approach. This project also uitilizes ASP.NET Identity to create/authenticate users and enforce security on the site

The site also contains admin features such ways to update lookup values, create revenue reports, and edit user roles. These features all rely on a well maintained and efficient SQLite database.
